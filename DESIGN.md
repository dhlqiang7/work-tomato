# Tomato - 个人工作跟踪与番茄钟系统 设计文档

## 1. 项目概述

### 1.1 定位

一款本地运行的**个人工作助理**，集成任务管理、番茄钟专注、工作回顾三大核心能力，帮助用户高效管理日常工作。

### 1.2 核心原则

- **纯本地运行**：所有数据存储在本地，不与任何网络服务交互
- **零配置启动**：一条命令启动，自动打开浏览器
- **界面优先**：现代化 UI，支持深色/浅色主题
- **安全可控**：仅使用商业友好协议（MIT/Apache-2.0/BSD）的开源依赖

## 2. 技术架构

### 2.1 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 运行时 | Node.js >= 18 | 唯一运行时依赖 |
| 后端 | Express.js | 轻量 HTTP 服务 |
| 前端 | Vue 3 (SPA) | 响应式组件化界面 |
| 样式 | 原生 CSS + CSS Variables | 自定义设计系统，支持主题切换 |
| 存储 | JSON 文件 | 本地文件系统，按模块拆分 |
| 构建工具 | Vite | 开发热更新 + 生产构建 |

```
  → 启动：初始化定时提醒调度器
  → 自动打开默认浏览器访问 http://localhost:3210
```

## 3. 数据模型

### 3.1 任务 (Task)

```jsonc
{
  "id": "uuid",                   // 唯一标识
  "title": "string",              // 任务标题（必填）
  "description": "string",        // 任务描述/备注
  "projectId": "string | null",   // 所属项目 ID，为空则归入默认项目
  "priority": "P0|P1|P2|P3",      // 优先级：紧急/高/中/低，默认 P2
  "status": "pending|active|done|archived",  // 状态
  "tags": ["string"],             // 标签列表
  "relatedPeople": ["string"],    // 关联人员
  "deadline": "ISO8601 | null",   // 截止时间
  "reminderDismissed": false,     // 提醒是否已屏蔽
  "reminderSnoozedUntil": null,   // 延迟提醒到何时
  // 番茄钟相关
  "estimatedPomodoros": 4,        // 预估番茄数（可选）
  "completedPomodoros": 0,        // 已完成番茄数
  "totalFocusMinutes": 0,         // 累计专注分钟数
  // 完成相关
  "completedResult": "string",    // 完成时补充的成果描述
  "completedAt": "ISO8601",       // 完成时间
  // 时间戳
  "createdAt": "ISO8601",
  "updatedAt": "ISO8601"
}
```

### 3.2 项目 (Project)

```jsonc
{
  "id": "uuid",
  "title": "string",              // 项目标题（必填）
  "description": "string",        // 项目描述
  "relatedPeople": ["string"],    // 关联人员
  "color": "#hex",                // 项目标识色
  "status": "active|archived",    // 状态
  "createdAt": "ISO8601",
  "updatedAt": "ISO8601"
}
```

### 3.3 番茄钟记录 (Pomodoro)

```jsonc
{
  "id": "uuid",
  "taskId": "string",             // 关联任务
  "type": "work|break|longbreak", // 类型：工作/短休息/长休息
  "plannedMinutes": 25,           // 计划时长
  "actualMinutes": 25,            // 实际时长
  "status": "running|completed|cancelled",  // 状态
  "startedAt": "ISO8601",
  "endedAt": "ISO8601 | null"
}
```

### 3.4 默认项目

系统启动时自动创建一个默认项目（ID: `default`，标题: "日常工作"），未指定项目的任务自动归入此项目。

## 4. 功能设计

### 4.1 任务管理

#### 4.1.1 新建任务

- **必填**：任务标题
- **选填**：所属项目（默认项目）、优先级（默认 P2）、截止时间、关联人员、标签、描述、预估番茄数
- 交互：支持键盘快捷键 `N` 快速新建

#### 4.1.2 任务列表

- 默认按优先级排序（P0 → P3），同优先级按截止时间升序
- 支持视图切换：列表视图 / 看板视图（按状态分列）
- 支持按项目、标签、优先级筛选
- 支持关键词搜索（标题 + 描述）
- 过期任务红色高亮标记

#### 4.1.3 编辑 & 删除

- 点击任务卡片进入编辑模式，支持修改所有字段
- 删除前弹出确认对话框
- 支持批量操作（多选后批量修改项目/优先级/删除）

### 4.2 项目管理

#### 4.2.1 新建项目

- **必填**：项目标题
- **选填**：描述、关联人员、标识色（提供预设色板）
- 展示项目下的任务计数和完成进度

#### 4.2.2 项目列表

- 卡片式展示，显示标题、任务统计、进度条
- 支持归档已完成项目（归档后不在活跃列表显示，但数据保留）

### 4.3 番茄钟

#### 4.3.1 经典模式（25/5）

- **工作阶段**：25 分钟倒计时
- **短休息**：5 分钟（每个番茄后）
- **长休息**：15 分钟（每 4 个番茄后）
- 配合环形进度条动画，视觉展示剩余时间

#### 4.3.2 操作流程

```
选择任务 → 点击"开始专注" → 25分钟倒计时（环形动画）
  → 倒计时结束 → 提示音 + 弹窗通知
  → 短休息 5 分钟 → 自动开始下一个番茄 / 或用户手动开始
  → 每 4 个番茄 → 长休息 15 分钟
```

#### 4.3.3 交互细节

- 倒计时期间可暂停/放弃
- 暂停超过 5 分钟自动记录为中断并提示是否继续
- 最小化到浏览器标签页标题显示剩余时间（如 `🍅 18:32 - 任务标题`）
- 标签页图标动态变化（工作中显示番茄，休息中显示咖啡杯）

### 4.4 完成工作

- 点击"完成任务"触发完成流程
- 弹出对话框，可补充**工作成果**描述
- 自动归档：记录完成时间，状态变更为 `done`
- 已完成任务进入"今日回顾"列表

### 4.5 回顾与总结

#### 4.5.1 时间维度

| 维度 | 说明 |
|------|------|
| 今日 | 当天完成的所有任务 |
| 本周 | 最近 7 天（含今天） |
| 本月 | 当前自然月 |
| 最近 30 天 | 滚动 30 天 |
| 半年度 | 当前自然年的上下半年 |
| 自然年 | 当前自然年 |
| 自定义 | 用户自选起止日期 |

#### 4.5.2 回顾内容

- 完成的任务列表（含成果描述）
- 统计数据：完成任务数、专注时长、番茄数
- 趋势图：每日专注时长折线图
- 按项目分布的饼图
- 按优先级分布的柱状图

### 4.6 工作提醒

#### 4.6.1 提醒规则

| 触发条件 | 提醒内容 |
|---------|---------|
| 任务截止时间是今天 | "【任务名】今天到期" |
| 距截止时间不足 1 小时 | "【任务名】将在 XX 分钟后到期" |
| 任务已过期 | "【任务名】已过期 XX 天" |

#### 4.6.2 提醒操作

- **开始工作**：直接启动该任务的番茄钟
- **延迟截止时间**：推迟 1 小时 / 推迟到明天 / 自定义时间
- **不再提醒**：标记 `reminderDismissed = true`

#### 4.6.3 提醒方式

- 浏览器内通知弹窗（右上角 toast）
- 若浏览器支持 Notification API，同时发送系统通知

### 4.7 统计仪表盘

- 今日完成数 / 总任务数
- 今日专注时长
- 当前连续工作天数（ streak ）
- 本周每日专注时长柱状图
- 热力图：最近 12 周的每日工作强度（类 GitHub 贡献图）

## 5. 补充功能

### 5.1 主题切换

- 浅色 / 深色主题，通过 CSS Variables 实现
- 跟随系统偏好自动切换，也可手动覆盖
- 主题偏好持久化到 localStorage

### 5.2 数据导入导出

- 一键导出全部数据为 JSON 文件（备份）
- 从 JSON 文件导入恢复（合并或覆盖）
- 导出时包含版本号，便于未来数据迁移

### 5.3 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `N` | 新建任务 |
| `P` | 暂停/继续番茄钟 |
| `S` | 开始番茄钟（当前选中任务） |
| `Esc` | 关闭弹窗 / 取消编辑 |
| `Ctrl/Cmd + F` | 搜索任务 |
| `?` | 显示快捷键帮助 |
| `1-4` | 切换侧边栏视图（任务/项目/番茄钟/统计） |

### 5.4 音效

- 番茄钟结束：清脆的铃声
- 休息结束：柔和的提示音
- 任务完成：轻微的成功音效
- 音效文件内置于 assets 目录，支持静音切换

### 5.5 聚焦模式

- 隐藏侧边栏和非必要 UI 元素
- 仅显示当前番茄钟 + 任务信息
- 适合专注工作场景

```
┌─────────────────────────────────────────────────┐

│          │                                      │
│  📋 任务  │         主内容区域                    │

│──────────│                                      │

│ ⚠ 即将过期│                                      │
```

### 6.2 设计风格

- **整体风格**：简洁现代，圆角卡片式布局
- **配色方案**：
  - 浅色模式：白色背景 + 灰色卡片 + 番茄红强调色（#E74C3C）
  - 深色模式：深灰背景 + 深色卡片 + 番茄红强调色
- **字体**：系统默认字体栈，中文优先
- **动画**：页面切换淡入淡出，番茄钟环形动画，列表操作过渡动画
- **间距**：8px 基础栅格，统一一致的间距体系

### 6.3 关键页面

1. **任务页**：列表/看板双视图，筛选栏 + 任务卡片列表
2. **番茄钟页**：大型环形倒计时 + 当前任务信息 + 今日统计
3. **统计页**：数据卡片 + 图表区域
4. **回顾页**：时间维度切换 + 任务列表 + 统计摘要

## 7. API 设计

### 7.1 任务 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/tasks` | 获取任务列表（支持 ?status=&projectId=&priority= 筛选） |
| GET | `/api/tasks/:id` | 获取单个任务 |
| POST | `/api/tasks` | 创建任务 |
| PUT | `/api/tasks/:id` | 更新任务 |
| DELETE | `/api/tasks/:id` | 删除任务 |
| POST | `/api/tasks/:id/complete` | 完成任务（body 中可传 completedResult） |
| GET | `/api/tasks/reminders` | 获取需要提醒的任务 |

### 7.2 项目 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/projects` | 获取项目列表 |
| POST | `/api/projects` | 创建项目 |
| PUT | `/api/projects/:id` | 更新项目 |
| DELETE | `/api/projects/:id` | 删除项目 |

### 7.3 番茄钟 API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/pomodoros/start` | 开始番茄钟（body: taskId, type） |
| PUT | `/api/pomodoros/:id/pause` | 暂停 |
| PUT | `/api/pomodoros/:id/resume` | 继续 |
| PUT | `/api/pomodoros/:id/stop` | 停止（完成或取消） |
| GET | `/api/pomodoros/current` | 获取当前进行中的番茄钟 |

### 7.4 统计 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/stats/dashboard` | 仪表盘数据 |
| GET | `/api/stats/review?period=today` | 回顾数据（period: today/week/month/month30/halfyear/year/custom） |
| GET | `/api/stats/export` | 导出全部数据 |
| POST | `/api/stats/import` | 导入数据 |

## 8. 非功能需求

### 8.1 性能

- 页面首次加载 < 2 秒
- 任务列表支持 1000+ 条目流畅滚动（虚拟滚动）
- JSON 文件读写加锁，防止并发数据损坏

### 8.2 可靠性

- 数据写入采用"写临时文件 → 重命名"策略，防止写入中断导致数据丢失
- 服务端全局错误捕获，避免未处理异常导致进程崩溃
- 前端 API 请求统一错误处理与重试

### 8.3 安全性

- 仅监听 localhost，不暴露到局域网
- 前端输入做 XSS 防护（转义 HTML）
- 无用户认证需求（纯本地单用户）
- 依赖包上线前审查许可证兼容性
